# Mastermind Game (C++)

This project is a simple console-based implementation of the Mastermind game written in C++.

## Game Description
The game is played by two players:
- One player creates a secret color combination.
- The other player tries to guess the combination.

After each guess, the program gives feedback:
- **R** → correct color in the correct position
- **W** → correct color but wrong position

The guesser has up to **6 attempts** to find the secret combination.

## Valid Colors
The allowed colors are:

R - Red  
G - Green  
B - Blue  
P - Purple  
Y - Yellow  
M - Magenta  

Each color can be used **only once** in the combination.

## Features
- Input validation for guesses and secrets
- Feedback system for guesses
- Multiple game runs
- Score tracking for both players
